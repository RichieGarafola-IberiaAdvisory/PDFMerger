# 📄 PDF Packet Builder (Streamlit)

A lightweight Streamlit application that allows users to upload multiple PDF files and merge them into a single downloadable packet.

---

## 🚀 Features

- Upload multiple PDF files
- Merge PDFs into a single document
- Instant download of merged packet
- Runs locally with no external dependencies or paid services
- Handles corrupted PDFs gracefully (skips invalid files)

---

## 🧱 Tech Stack

- Python
- Streamlit
- pypdf
